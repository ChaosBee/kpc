使用size属性可以定义图标尺寸，组件内置了4中尺寸: large default small mini。你可以通过数字或这数字字符串自定义图标的大小

index.vdt index.styl index.js index.vue (Vue)index.jsx (React)
颜色

使用color属性可以定义图标的颜色，组件内置了4中颜色：primary warning danger success。你也可以自定义color值

index.vdt index.styl index.js index.vue (Vue)index.jsx (React)

属性

属性 说明 类型 默认值
size 定义图标尺寸 "default" | "large" | "small" | "mini" | String | Number "default"
color 图标颜色 "default" | "primary" | "danger" | "warning" | "success" | String undefined
