基础用法

通过添加to属性，来展示超链接

index.vdt index.js index.vue (Vue)index.jsx (React)

指定分隔符

通过separator属性或者separator扩展点可以指定分隔符

index.vdt index.styl index.js index.vue (Vue)index.jsx (React)

属性

Breadcrumb

属性 说明 类型 默认值
separator 指定分隔符 String undefined

BreadcrumbItem

属性 说明 类型 默认值
to 指定超链接地址 String undefined
separator 指定分隔符，会覆盖Breadcrumb中的定义 String ">"

扩展点

Breadcrumb

名称 说明
separator 扩展分隔符

BreadcrumbItem

名称 说明
separator 扩展分隔符，优先级高于Breadcrumb中的定义
