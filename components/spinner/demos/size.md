## zh-CN

`size`属性定义尺寸：`large`, `default`, `small`, `mini`

## en-US

The `size` attribute defines the size: `large`, `default`, `small`, `mini`

```vdt
import Spinner from 'kpc/components/spinner';

<div>
    <Spinner size="large" />
    <Spinner />
    <Spinner size="small" />
    <Spinner size="mini" />
</div>
```

```styl
.k-spinner
    margin-right 20px
```
